# 🐾 Pet Paradise Web Application

## 📌 Project Overview
Pet Paradise is a full-stack Java web application developed for online pet browsing and purchasing.  
The system allows users to explore pets, add them to cart or wishlist, place orders, and make secure payments.  
It also includes an admin panel for managing pets, orders, and customer activities.

---

## 🚀 Features
- User Registration & Login
- Browse Pets by Category
- Add to Cart & Wishlist
- Secure Checkout System
- Order Management
- Admin Dashboard
- Manage Pets
- Manage Orders
- Responsive UI Design

---

## 🛠️ Technologies Used
- Java
- JSP & Servlet
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat
- Maven

---

## 🗄️ Database
MySQL database is used for storing:
- User details
- Pet details
- Orders
- Cart items
- Wishlist items
